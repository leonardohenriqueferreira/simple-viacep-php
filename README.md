# Simple ViaCEP PHP

Busca de CEP utilizando o webservice [ViaCEP](https://viacep.com.br/)

## Install

```
$ composer require leonardoferreira/simple-viacep
```

## Usage

```
SimpleViaCep::search("01001000");
```
